# Индексация и SEO-мониторинг (этап 0.15–0.18)

> Дата проверки: 2026-06-25  
> Сайт: https://deconstruction.ru

## 0.15 Индексация (Яндекс / Google)

### Google

По запросу `site:deconstruction.ru` страницы индексируются. В выдаче присутствуют:

- Главная (`/`)
- Услуги (`/uslugi/demontazh/...`)
- Блог (`/demontazhnyy-blog/...`)
- Кейсы и гео-страницы

**Вывод:** сайт в индексе Google, основные разделы доступны.

### Яндекс

Требует входа в [Яндекс.Вебмастер](https://webmaster.yandex.ru/) для точного числа страниц.  
Экспорт Tilda содержит `robots.txt` и `sitemap.xml` — оба совпадают с продакшеном.

**Рекомендация при миграции:** после релиза отправить новый sitemap в Вебмастер и запросить переобход.

## 0.16 Карта редиректов

- **349** ЧПУ → `page*.html` (внутренние rewrite)
- **117** внешних 301-редиректов (старые URL → новые)
- Полный JSON: `_data/audit.json → redirects301`

Критичные редиректы для Next.js `next.config.ts`:

- `/portfolio` → `/nashi-proekty/`
- `/arenda-robotov-brokk-*` → `/arenda-demontazhnykh-robotov/...`
- `/primery-rabot/*` → `/` (многие на главную)
- `/nashi-uslugi/*` → `/uslugi/` или `/`

## 0.17 Страницы с трафиком (Метрика)

**Требует доступа к Яндекс.Метрике.**

После получения доступа выгрузить отчёт «Содержание → Страницы входа» за 12 месяцев и сопоставить с `src/content/pages.ts`.

Приоритет при переносе — страницы с органическим трафиком > 100 визитов/мес.

## 0.18 Позиции по запросам

**Требует доступа к Serpstat / Topvisor / Search Console.**

Ориентировочные целевые кластеры (из контента сайта):

| Кластер | Примеры запросов |
|---------|------------------|
| Демонтаж роботами | демонтаж роботом brokk, аренда brokk |
| Демонтаж зданий | демонтаж зданий спб, снос зданий москва |
| Услуги | демонтаж бетона, демонтаж фундамента, алмазная резка |
| Гео | демонтаж спб, демонтаж москва, районы СПб |

Зафиксировать текущие позиции **до релиза** и сравнить через 2–4 недели после.
